## Key Concepts

### Version Control System (VCS)
A system that tracks changes in files over time, allowing multiple people to collaborate.

### Git
- **Distributed**: Every developer has a complete copy of the history
- **Fast**: Operations are performed locally
- **Reliable**: Cryptographic integrity checking
- **Flexible**: Supports many workflows

### Repository
A directory tracked by Git containing your project files and version history.

### Commit
A snapshot of your changes with a message describing what changed and why.

### Branch
A parallel line of development in your repository.
