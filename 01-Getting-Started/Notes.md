# 📌 Chapter 1 Quick Notes

## Git Definition
Version control system to track code changes.

## Creator
Linus Torvalds (2005)

## Types of VCS
1. Local
2. Centralized (SVN, CVS)
3. Distributed (Git)

## Git Key Idea
Stores snapshots, not differences.

## Git Benefits
- Fast
- Distributed
- Safe
- Offline work possible

## Git States
- Modified
- Staged
- Committed

## Workflow
Edit → Add → Commit → Push

## Integrity
Uses SHA-1 hashing for safety

## Setup Commands
git config --global user.name
git config --global user.email
