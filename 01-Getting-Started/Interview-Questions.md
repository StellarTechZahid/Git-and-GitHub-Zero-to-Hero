# 💼 Interview Questions

## What is Git?
A distributed version control system used to track code changes.

## Who created Git?
Linus Torvalds in 2005.

## What is DVCS?
A system where every user has full repository copy.

## Difference between Git and SVN?
Git is distributed, SVN is centralized.

## What are Git states?
- Modified
- Staged
- Committed

## Why is Git fast?
Because it works locally.

## What is a commit?
A snapshot of project at a specific time.
