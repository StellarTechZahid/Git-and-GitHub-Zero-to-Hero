# Intermediate Git & GitHub 🚀

Master branching strategies, merging techniques, and GitHub collaboration.

## Module Overview

This module covers:
- Branching strategies (Git Flow, GitHub Flow, Trunk-Based)
- Merging and rebasing techniques
- Handling merge conflicts
- GitHub pull requests and code reviews
- Collaboration best practices
- GitHub features for teams
