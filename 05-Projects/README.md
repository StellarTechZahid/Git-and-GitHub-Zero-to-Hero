## General Requirements

All projects should include:

✅ Professional README with setup instructions
✅ Clear commit messages following conventions
✅ Proper branch strategy
✅ Documentation
✅ Code of Conduct
✅ Contributing guidelines
✅ License file
✅ Good git history (no squashing except where appropriate)

## Submission

To submit your project:

1. **Create a new branch**: `feature/project-submission`
2. **Add project documentation**: In projects folder
3. **Create pull request**: With detailed description
4. **Include**: Screenshots, links, results
5. **Submit**: Share PR link
